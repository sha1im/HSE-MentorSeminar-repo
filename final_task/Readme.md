## Запуск сервиса локально

### Установка зависимостей

Перед запуском необходимо установить зависимости. Лучше всего сделать это в виртуальном окружении.
Зависимости устанавливаются с помощью команды:

pip install -r requirements.txt

### Запуск сервера

Из корня проекта выполните команду:

uvicorn todo_service_app.main:app

### Доступ к сервису

После запуска сервер будет доступен по адресу:

http://localhost:8000


База данных SQLite создаётся автоматически при первом запуске.

## Запуск сервиса через Docker

### Сборка Docker-образа

Из корня проекта выполните:

docker build -t todo-service:local .

Создание volume для хранения данных:
docker volume create todo_data

### Запуск контейнера

docker run --rm -p 8080:80 -v todo_data:/app/data --name todo-service todo-service:local

### Доступ к сервису

После запуска контейнера сервис будет доступен по аналогичному адресу:

http://localhost:8080

Переменная PATH является переменной окружения текущего процесса bash.
При редактировании подобной переменной, изменения акутальны для текущего окружения.
Когда мы закрываем терминал, процесс завершается и окружение исчезает. При последущем открытии наши изменения уже не будут видны.
Для того, чтобы сделать изменения постоянными, следует изменить файл ~/.bashrc. Данный файл читается при каждом запуске shell и является, по сути, последней настройкой терминала при запуске.
Внеся изменения в данный файл, мы сделаем их постоянными.
Изменения можно внести дописав следующую команду:
export PATH+=":<your directory>"
